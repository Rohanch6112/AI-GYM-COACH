# 🏋️ AI Gym Coach

AI Gym Coach is an AI-powered virtual fitness assistant that uses Computer Vision and Pose Detection to analyze workouts in real time. The application tracks exercises, counts repetitions, monitors posture accuracy, and provides intelligent coaching feedback through AI-driven interactions.

Built with Python, Streamlit, OpenCV, and MediaPipe, this project demonstrates real-time human pose estimation, exercise recognition, and interactive AI-based fitness coaching.

---

## 🚀 Features

- 🎯 Real-time pose detection
- 💪 Exercise repetition counting
- 🧍 Posture correction and form analysis
- 🤖 AI-powered workout coaching
- 🎤 Voice feedback support
- 📹 Live webcam integration
- 📊 Workout tracking and monitoring
- 🔐 User authentication system
- ⚡ Interactive Streamlit UI
